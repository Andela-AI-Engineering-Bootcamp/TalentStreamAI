# TalentStreamAI frontend

Next.js 15 (App Router) client for the capstone. Local configuration is loaded from the **repository root** `.env` via `dotenv-cli` in the npm scripts so the UI stays aligned with the FastAPI service.

## Commands

```bash
npm install
npm run dev
npm run lint
npm run build
```

For full-stack instructions (Docker, GitHub deploy, Terraform), read the root `README.md`.
